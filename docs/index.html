✦ CAPABILITIES

What JARVIS Can Do

Over 20 built-in tools let JARVIS control your computer, browse the web, create documents, and more — all through natural conversation.

🧠

Natural Chat

Conversational AI with memory across sessions. Remembers facts, recalls past conversations, understands context.

🖥️

Desktop Control

Open apps, manage windows, adjust volume & brightness, take screenshots, lock, sleep, shutdown — all by voice or text.

🌐

Web & Search

Open websites, search Google, play YouTube, check Wikipedia, get real-time weather — directly from chat.

📁

File Management

Create, read, edit, move, copy, search, and delete files anywhere on your disk. Protected system directories are blocked.

📄

Document Creator

Generate PDF, Word, Excel, PowerPoint, Markdown, HTML, CSV, images — from simple natural language descriptions.

🧮

Math Engine

Solve equations, derivatives, integrals, algebra — powered by SymPy. Fast, accurate, no internet needed.

📱

Telegram Remote

Full remote control from anywhere via Telegram. Inline keyboards, approval flow, push alerts, memory backup.

💬

WhatsApp Messaging

Send messages and files via WhatsApp Desktop with automated UI. Works with any contact in your WhatsApp.

🎯

Autonomous Goals

Set multi-step goals with cron scheduling. JARVIS plans, executes, and reflects on tasks independently.

🎭

Multiple Personas

Choose from JARVIS (British butler), FRIDAY (casual friend), or COACH (productivity mode). Each with unique tone.

🎨

7 Themes

Arc, Mark III, Ultron, Stealth, Midnight, AMOLED, Light — a full glassmorphism design system with live switching.

🖼️

Vision & Screen

Describe images, analyze screenshots, read what's on your screen — built-in multimodal AI vision.

🏗️ ARCHITECTURE

How It Works

Python + Google Gemini + Flask — all running locally on your machine.

🧠 Google Gemini

Brain of the operation. Handles natural language understanding, tool selection, vision, and conversation. Free tier with unlimited usage.

⚙️ Python Backend

Flask web server with threaded request handling. Tool registry with 20+ automations. Background scheduler for cron goals.

🎨 Web HUD

Vanilla HTML/CSS/JS with no frameworks. Canvas-based arc reactor, glassmorphism design, real-time SSE streaming.

📱 Telegram Bot

Async Python bot with inline keyboards, approval flow, push alerts. Runs as a daemon thread inside the Flask server.
